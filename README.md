# booking-app

### Description

Booking App is a demo project that showcases a simple service for authentication and booking management.

### Technologies

Backend: Node.js, TypeScript, Express, MongoDB (configured to run in Docker).

Frontend: Flutter (Dart, BloC, Dio) â€“ a mobile app for authentication, viewing, and creating services and bookings via REST API.

### Installation

1. Start the Backend
```sh
cd backend
npm install
npm run build
npm run start:docker   # Run in Docker
```
2. Start the Frontend
```sh
cd frontend
flutter pub get
flutter run
```
